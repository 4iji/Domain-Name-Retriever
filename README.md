## README ##

The Aim of this project is to predict the domain name, given the name of the company.

Input: Company Name
Output: Domain of the company.

examples:


1) Microsoft
    microsoft.com

2) Split an Atom
    splitanatom.com

3) Guru Technologies
    getguru.com

4) Outwork
    outwork.com

5) Total Pest
    totalpest.co
